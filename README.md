# Ethos-Questionario-Microserviço

Microserviço desenvolvido em Java, abrangendo todos os endpoints relacionados aos questionários no projeto Ethos. Destaca-se pela arquitetura modular e funcionalidades específicas.

## Funcionalidades Principais

- **Abrangência Total:** Todos os serviços relacionados aos questionários no Ethos são contemplados pelos endpoints deste microserviço.

Certifique-se de explorar cada detalhe, e sinta-se à vontade para contribuir ou derivar funcionalidades conforme necessário. Este microserviço de questionários desempenha um papel crucial na arquitetura do projeto Ethos, proporcionando uma base robusta para as operações relacionadas aos questionários estabelecidos.
